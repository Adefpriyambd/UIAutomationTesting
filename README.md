## How to use this code
### Installization
```
npm init
npm install
npm init wdio .
npm install wdio-cucumberjs-json-reporter --save-dev
npm install multiple-cucumber-html-reporter --save-dev
```
### How to run
```
npm run wdio
```
### How to create and open report
```
start .tmp\report\index.html
```
